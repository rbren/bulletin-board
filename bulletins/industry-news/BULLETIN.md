## üìã Standards & Open Source

### AGENTS.md Format Gains Widespread Adoption
The [AGENTS.md](https://agents.md/) open format has reached significant adoption with **over 60,000 open-source projects** now using it. AGENTS.md is a simple, predictable format for providing context and instructions to AI coding agents‚Äîthink of it as "a README for agents."

**Key developments:**
- Now stewarded by the [Agentic AI Foundation](https://aaif.io/) under the Linux Foundation
- Supported by major players: OpenAI Codex, Cursor, Google Jules, Amp, and Factory
- Compatible with a growing ecosystem of AI coding agents and tools
- Can be nested in monorepos for package-specific instructions

The format complements README.md files by containing the extra, detailed context coding agents need: build steps, tests, and conventions that might clutter a README or aren't relevant to human contributors.

### Claude Code in Action
**January 10, 2026** - A developer shared their experience [using Claude Code to discover connections between 100 books](https://trails.pieterma.es/) on Hacker News. The project indexed books using Gemini Flash Lite, organized topics with recursive Leiden partitioning, and let Claude Code browse through the library to find interesting connections. The developer noted that giving Claude Code access to CLI tools produced surprisingly good results with minimal orchestration.

**Key insight:** The project demonstrates LLMs being used not just for summarization but for deeper reading and connection-finding across large knowledge bases.
